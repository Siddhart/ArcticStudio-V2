Rocket Launch Tracker

Description:

The Rocket Launch Tracker will get the latest planned rocket launches and it will send a message in your channel when a new launch was added.

Commands:

Here you can find a table with all my commands.

Command What does it do?
RT track Get info about the recently added launch.
RT get [number from 1 to 5] Get the latest added launches.
RT start Setup the notifications in your channel (only for users with admin permissions).
RT stop Stop the notifications in your server. (only for users with admin permissions).
RT help Get all the commands you can use.
