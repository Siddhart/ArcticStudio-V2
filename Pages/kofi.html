Setup the Ko-fi Bot
Ko-fi Discord Bot

Integrate Ko-fi in your Discord server.

by Arctic Studio

This video shows how you can setup the Discord bot.




You can choose between one of these themes.



1. Setup The Bot :
Invite the bot in your server :

Click here to get the invite link.

Make sure that you are the owner or that you have admin permissions in the server you want to use the bot in.

After that Select your server with the dropdown menu.

Click on the "continue" button and complete the recaptcha.


2. Setup the bot in your Discord channel :
Use the setup commands :

After you invited the bot you can use the command kofi help to get all the commands.

Type kofi setup in the channel you want the notifications in.

You can now use kofi themes to get all the themes and assign a theme with kofi settheme NUMBER.

Type kofi api to get the api link. KEEP THIS A SECRET.


3. Setup the webhook link on your Ko-fi api dashboard :
Setup the webhook :

Copy the api-url that the bot send in the channel. Make sure to not copy spaces.

Go to the api dashboard on the Ko-fi page.

Insert the link in the inputfield and click the 'Update' button. After this you can click the "Send Test" to test the request.

After you clicked the "Send test" button the bot will respond with a test message in the assigned channel.

If the bot didn't respond with a test message please join our support server and ask a developer for help.
