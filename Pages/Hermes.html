This bot will send a message everyday with a random event that happened in the past on the same day.

Commands:

Here you can find a table with all my commands.

Command What does it do?
HB get Type this command to get a random event that happend in the past on the same day.
HB setup Type this command in the channel where you want the daily messages.
HB stop Type this command to stop the daily messages.
HB help Type this command to get the all the commands.
