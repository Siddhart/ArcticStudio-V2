Free Games Bot

Description:

The Free Games Bot will automatically send a message in your server when a new game is free.
The bot is constantly looking for new games. If you want to test the bot join our Discord server.

Commands:

Here you can find a table with all my commands.

Command What does it do?
FG help Get all the commands.
FG get Get all the current Free Games.
FG setup Type this command in the channel where you want the bot to send the notifications.
FG role @ROLE Type this command to make the bot mention a specific role when a new game is free.
FG stop Type this command if you want the bot to stop sending messages in the assigned channel.
