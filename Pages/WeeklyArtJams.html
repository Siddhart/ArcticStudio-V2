Weekly Art Jams

Description:

This bot was made for people with art based Discord servres.
Use this bot to setup weekly art jams in your server.
It will send a random theme every monday at 12:00 CEST.

Commands:

Here you can find a table with all my commands.

Command What does it do?
TB help Get all the commands that you can use.
TB start Use this command to assign the current channel.
TB stop Use this command to stop the bot from sending messages.
TB suggest YOUR THEME Suggest your theme for the upcomming art jams.
